📖 User Manual & Help

🏠 1. Dashboard

Log daily expenses using the "Add New Expense" card. Select a category, enter the amount, and click Add.

📅 2. Calendar

View expenses by date. Click any date to add an expense directly for that day.

💳 3. Payments & Smart SMS

Pro Tip: Paste your bank's UPI SMS into the "Smart UPI Reader" box and click Extract to auto-fill the payment form!

⚙️ 4. More Options

  • Export: Download reports as Excel, PDF, Word, or CSV (for Google Sheets).
  • Backup: Save your data to a file to prevent data loss.

🔑 5. Passkey Login

Go to More Option to generate a 6-digit Passkey. Use it on the login screen for instant access!
